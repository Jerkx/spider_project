#How To Install On Termux
```
pkg update -y && pkg upgrade -y
pkg install git
git clone https://github.com/Jerkx/spider_project
cd spider_project
chmod +x *
bash spider.sh
```

#How To Install On Linux
```
apt-get update -y && apt-get upgrade -y
apt-get install git
git clone https://github.com/Jerkx/spider_project
cd spider_project
chmod +x *
bash spider.sh
```
